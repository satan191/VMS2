# BQ-Pro VMS 部署教程

## 🚀 使用 Netlify 部署（推荐）

### 第一步：准备工作
✅ 项目已经构建完成，`dist` 文件夹已生成

### 第二步：注册 Netlify 账号

1. 打开浏览器，访问：https://www.netlify.com/
2. 点击右上角 "Sign up"（注册）按钮
3. 选择注册方式（推荐用 GitHub 或 Google 账号登录）
4. 完成注册后会进入控制面板

### 第三步：部署网站

**方法A：拖拽部署（最简单）**

1. 在 Netlify 控制面板中，找到中间的大框框，上面写着 "Want to deploy a new site without connecting to Git? Drag and drop your site output folder here"
2. 打开文件管理器，找到：`d:\project\监控平台\dist` 文件夹
3. 直接拖拽整个 `dist` 文件夹到 Netlify 页面的框框里
4. 等待上传完成（通常10-30秒）

**方法B：点击上传**

1. 点击 "Add new site" → "Deploy manually"
2. 点击 "Browse to upload"
3. 选择 `d:\project\监控平台\dist` 文件夹
4. 点击上传

### 第四步：获取网址

上传完成后：
1. Netlify 会自动生成一个随机网址，类似：
   ```
   https://stunning-cupcake-abc123.netlify.app
   ```
2. 点击这个网址，你就能看到你的监控平台了！
3. 把这个网址发给别人，他们就能访问了！

### 第五步：自定义网址（可选）

如果你觉得随机网址太长或不好记：

1. 在 Netlify 控制面板，点击 "Site settings"
2. 点击 "Change site name"
3. 输入你想要的名字，比如 `bq-pro-vms`
4. 网址就会变成：`https://bq-pro-vms.netlify.app`

## 🔄 如何更新网站？

如果你修改了代码，想更新线上的网站：

1. 在项目文件夹中，重新构建项目：
   ```bash
   npm run build
   ```

2. 在 Netlify 控制面板，找到你的网站
3. 点击 "Deploys" 标签页
4. 拖拽新的 `dist` 文件夹到页面中
5. 等待几秒钟，网站就更新了！

## 💡 重要提示

1. **只需要上传 `dist` 文件夹**
   - 不要上传整个项目文件夹
   - 只上传 `dist` 文件夹里的内容

2. **网址是永久有效的**
   - Netlify 的免费版会一直保留你的网站
   - 除非你主动删除

3. **访问速度**
   - Netlify 在全球有CDN节点
   - 国内访问可能需要等几秒钟
   - 如果需要更快的国内访问速度，可以考虑用国内的托管平台

4. **Logo图片**
   - 如果你想在部署后看到自定义Logo
   - 记得先把 Logo 图片放到 `public` 文件夹
   - 然后重新运行 `npm run build`
   - 再上传新的 `dist` 文件夹

## ❓ 常见问题

**Q1：别人访问网址时，能看到我的代码吗？**
A：不能。他们只能看到网页界面，看不到你的源代码。

**Q2：网站能一直免费使用吗？**
A：是的！Netlify 的免费版本对于个人项目完全够用。

**Q3：我可以绑定自己的域名吗？**
A：可以！在 Netlify 设置中可以绑定自定义域名（需要你有一个域名）。

**Q4：如果我修改了代码怎么办？**
A：重新运行 `npm run build`，然后上传新的 `dist` 文件夹。

**Q5：网站打不开怎么办？**
A：检查是否上传的是 `dist` 文件夹，不是整个项目文件夹。

如果部署过程中遇到问题，可以：

2. 确认上传的是 `dist` 文件夹的内容
3. 查看 Netlify 的部署日志，看是否有错误信息
